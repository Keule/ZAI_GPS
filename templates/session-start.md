# Session Start

## Zweck
Kurzstart für eine neue Arbeitssession mit klarem Scope und Erfolgskriterium.

## Minimalpflichtfelder
- **Datum/Zeit (UTC):**
- **Bearbeiter:**
- **Kontext/Issue/Link:**
- **Ziel der Session (1 Satz):**
- **Definition of Done (DoD):**
- **Erster geplanter Schritt:**

## Vorlage
```md
# Session Start
- Datum/Zeit (UTC):
- Bearbeiter:
- Kontext/Issue/Link:
- Ziel der Session (1 Satz):
- Definition of Done (DoD):
- Erster geplanter Schritt:
```

## Beispiel
```md
# Session Start
- Datum/Zeit (UTC): 2026-04-17 09:00
- Bearbeiter: Max Mustermann
- Kontext/Issue/Link: #42 „CRC-Handling Discovery"
- Ziel der Session (1 Satz): Discovery- und Core-CRC-Pfade testbar trennen.
- Definition of Done (DoD): PR mit grüner Matrix + aktualisierte Doku.
- Erster geplanter Schritt: Bestehende CRC-Entscheidungslogik in `dependency_policy` prüfen.
```
